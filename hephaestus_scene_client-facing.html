Knauf VR Training Academy — scene overview

Meta Quest VR · two content modes (KTA product showroom / VPT Virtual Plant Tour) · switched by a build-time toggle
This diagram maps the structure of the Knauf VR Training Academy app for Meta Quest. Each coloured box is a space or environment within the experience, and the arrows show how players move between them. The app runs in one of two modes — the KTA product showroom or the VPT factory tour — determined by a configuration switch set at build time. Both modes share the same startup sequence but branch into entirely separate environments from there.

Abbreviations & terms
KTA Knauf Training Academy — the product showroom mode. Players explore Knauf insulation product ranges in a virtual showroom environment.
VPT Virtual Plant Tour — the factory walkthrough mode. Players follow the real Knauf glass-wool manufacturing process from raw materials through to finished product.
XR portal An in-VR doorway. Walking through it transports the player to the next space — the primary navigation method in the factory tour.
Facility Map An in-VR menu that lets players jump directly to any factory room without walking through all the rooms in sequence.
Boot The app startup phase — runs before any visible content loads. It checks a configuration switch to decide which mode (KTA or VPT) to launch.
Master hub A background scene (shown as diamonds) that acts as a traffic controller — it doesn't contain visible environments itself, but manages which room loads next and coordinates transitions.
⚠ Orphan scene A scene that exists in the project but is not connected to any part of the app — not reachable by players and not included in any build. Flagged for cleanup.
Connected external services
Analytics Mixpanel tracks how players move through the experience — which rooms they visit, how long they spend in each, and where they exit. This data is used to measure engagement and identify drop-off points without requiring any action from the player.
Access code gate The factory tour requires a valid access code before entry. Codes are validated by a server-side system, so they can be issued, changed, or revoked at any time without releasing a new version of the app.
Remote content Factory tour rooms are downloaded to the headset from Knauf's cloud servers rather than bundled inside the app. This keeps the app install size small and means room content (models, textures, narration) can be updated without going through the app store.
VR hardware Built exclusively for Meta Quest. The app uses Meta's XR SDK for controller tracking, haptic feedback (controller vibration on interactions), and in-editor simulation for testing without a physical headset.
Known issue
⚠ 01_StartingRoom — missing from remote content delivery. This is the main entry point of the factory tour and every player passes through it, but it was never added to the remote content system. Unlike all other tour rooms, it cannot be updated without a full app release. The development team has flagged this as a known gap to be resolved in a future update.
